    • a:显示所有程序
    • u:以用户为主的格式来显示
    • x:显示所有程序，不以终端机来区分
  • grep 过滤特定行
  • grep -v 去除特定行
  • awk 提取进程ID
kill -9  进程号
  • kill 停止进程
