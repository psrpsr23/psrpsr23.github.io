序列拼接的基本原理

序列拼接

  • 片段组装问题
    • 给定一组取自特定字母表的字符串集合F，寻找一个最短的字符串s，使得F中的每一个字符串都是s的一个连续子串。![[Pasted image 20231228162058.png]]
    • 主要问题：
      • 碱基错误
      • 不知道片段方向
      • 存在重复区域
      • 缺少覆盖
  • 序列片段组装模型
    • 步骤
        1. 首先进行序列片段的两两比较，确定可能的片段之间的覆盖
        1. 确定所有片段统一的覆盖模式，即确定各个序列片段的相对位置
        1. 最后确定片段组装结果，即确定目标序列
    • 最短公共超串模型
      • 给定一个字符串集合F，求出一个最短的字符串S，使得对于所有属于F的字符串f，S是f的超串。
    • 重建模型
      • 考虑到片段的误差和未知方向的问题
      • 假设f、g代表两条序列，f作为g近似子串的代价$$d_{s}(f,g)={mind(f,s)}_{s\in S(g)}$$
        • S(g)代表所有子串的集合
        • d为编辑距离
          • 匹配为0
          • 其他为1
        • 主要思想
          • 寻找一个尽可能短的字符串S，使每一个f或其反向互补串成为S的近似子串
    • 多重连续区模型
      • 如果一组序列片段最弱连接的交叠长度至少为t，称这组片段的多重序列比对是t-contig。
      • 给定一个片段集合和一个整数t，将F分割为最小数目的子集$C_{i}$每个$C_{i}$允许一个t-config
  • 片段序列覆盖图
    • 多图：两个顶点之间可能存在多条边的图
    • 覆盖多图：
      • F是一个有向图，每个顶点代表一个字符串
      • 如果f的t个字符的后缀与g的t个字符的前缀相同，则图中存在一条权值为t的有向边![[Pasted image 20240102185458.png]]
    • 求最短的公共超串等价于取权值最大的哈密顿路径
      • 哈密顿路径：在一个图中沿边访问每个顶点恰好一次的路径
    • 一般步骤![[Pasted image 20240102185710.png]]
      • 贪婪算法
        ![[Pasted image 20240118204932.png]]![[Pasted image 20240118204921.png]]

基因组组装

  • 基因组复杂度及其评估
    • 认识基因组
    • 主要评估指标
      • 基因组大小
      • 重复序列比例
      • 基因组杂合度
      • 染色体型和倍性
    • 基于二代测序数据的基因组复杂度估计
      • kmer分析
        • kmers深度：L-K+1
          • L：序列总长度
          • K：kmer长度
        • 通过kmers深度的分布规律来估计基因组的一些基本信息。如基因组大小、杂合度、重复序列比例等
      • 估计基因组大小（）
        • G大小的基因组有N段，长度为L
  • 基因组组装
    • 主要挑战
      • 测序错误
      • 重复序列
      • 单倍型差异
      • 超大基因组
    • 主要流程
      • 评估基因组复杂度，确定测序方案 kmer
      • 从头组装得到contig序列 OLC和DBG
      • 连接config，得到scaffold
      • 挂载染色体序列 HiFi
      • 评估基因组组装质量

确定测序方案

  • 鸟枪法的测序策略
    • 逐步克隆法![[Pasted image 20240102195706.png]]
      • 构建文库繁琐、费时费力，成本非常高
      • 对组装算法要求低
    • 全基因组鸟枪法![[Pasted image 20240102195730.png]]
  • 待测基因组复杂度评估
  • 测序技术（1,2,3代）
    • 基因组复杂度
    • 测序成本
    • 样品与测序文库制备的难易度
  • 测序文库片段的大小数量
  • 测序深度
    • 测序平均深度——基因组平均每个点位上被测序的次数

确定组装方案

  • 从头组装算法 ![[Pasted image 20240102201112.png]]
  • ![[Pasted image 20240113135941.png]]
    • DBG算法（二代测序算法）
      • 哈尼尔顿回路（从某个起点开始，经过每个顶点一次且仅一次 ，回到起点）
      • 欧拉回路（从某个起点开始，经过每条边一次且仅一次，回到起点）【容易】
      • 在de Bruijn图中寻找一条欧拉超路能遍历所有的读段对应的路径
      • 德布鲁因序列B(k,n)，是k个元素构成的循环序列。所有长度为n的k元素构成序列都在它的子序列中，出现并且仅出现一次
      • 可以通过确定n维德布鲁因图的汉密尔顿路径
        • 构造de Bruijn图，每一个节点赋值3-mer
        • 连接节点，边的取值为4-mer，它的前缀和后缀分别代表了两个节点的3-mer
    • OLC算法
      • Sanger算法
        • 找出序列间的重叠区域
        • 基于重叠区域，拼接出序列片段
        • 基于序列片段关系进行连接，经错误纠正，获得最终序列
  • scaffolding
    • 按照一定的顺序和方向，将基因组上物理位置相邻的contig序列，连接成scaffold序列
    • Contig/Scaffold N50/L50 将组装获得的contig/scaffold 序列按长度从大到小依次累加，当累计长度超过contig/scaffold总长度的50%时，此刻对应的那个contig/scaffold的长度即为N50值，此时被累加的contig/scaffold数目即为L50值。
