序列比对简介

  • 序列比对：根据特定的计分规则，通过一定的算法对两条或多条DNA或蛋白质序列进行比较，找出他们之间的最优匹配或最大相似度匹配
    • 根本任务
      • 发现序列之间的相似性
      • 辨别序列之间的差异
    • 目的
      • 相似序列 -> 相似结构 -> 相似功能
      • 判别同源性
      • 推测序列之间的进化关系
      • 寻找遗传变异
    • 核心问题
      • 用于评估序列比对的评分系统
      • 用来寻找最佳比对的算法
      • 用来评估对比得分显著性的统计方法

序列比对的计分规则

  • 最优匹配
    • 在给定计分规则下，打分最高或罚分最低的比对形式
    • 匹配 0
    • 错配 -1
    • 空位 -1
  • 记分矩阵
    • 序列比对过程中使用的计分规则，是序列比对的重要组成部分，他给出序列比对中碱基或氨基酸匹配或错配值，故又称替换矩阵
      • 生物学关系最显著的对比形式的统计值最大
      • 长度一定时，分数越高，两条序列匹配越好
    • DNA替换矩阵
      • DNA替换矩阵相对简单
      • PAM1=每个位置有1%的突变概率![[Pasted image 20231220124010 1.png]]
      • 考虑转换(Transitions)比颠换(Transversions)更频繁![[Pasted image 20231220124231.png]]
        • 核苷酸的碱基按照环结构特征被划分为两类，一类是嘌呤，包括腺嘌呤A和鸟嘌呤G（两个环）；另一类是嘧啶，包括胞嘧啶C和胸腺嘧啶T（一个环）。如果DNA 碱基的替换保持环数不变，则称为转换，比如腺嘌呤A 替换为鸟嘌呤G、或者胞嘧啶C替换为胸腺嘧啶T，也就是嘌呤变嘌呤，嘧啶变嘧啶；如果环数发生变化，则称为颠换，比如腺嘌呤A 替换为胞嘧啶C、或者胸腺嘧啶T 替换为鸟嘌呤G，也就是嘌呤变嘧啶，或者嘧啶变嘌呤。在进化过程中，转换发生的频率远比颠换高。也就是说，大自然更倾向于接受嘌呤和嘌呤之间的替换，以及嘧啶和嘧啶之间的替换，而嘌呤和嘧啶之间的替换会导致不好的事情发生，这种替换大多在进化过程中已经被淘汰了。
      • 如果一个替换的概率是2%，只需要将1%的概率乘以自己【矩阵乘法】
      • PAM N矩阵式通过PAM 1自乘N次
    • 氨基酸替换矩阵
      • 氨基酸替换矩阵相对复杂
        • 不同错配的打分处理
        • 一致氨基酸的计分不同
        • 稀有氨基酸分高，普通氨基酸分低
        • 结构或性质相似的氨基酸之间的替换很常见
        • 不同蛋白质接祖和预期的相似度构建不同的替换矩阵
      • Dayhoff矩阵
        • 基于蛋白质进化模式的研究
        • 71组蛋白质，>85%序列一致性，1572次氨基酸替换事件![[Pasted image 20231220125147 1.png]]
        • ![[Pasted image 20231220125257.png]]
        • ![[Pasted image 20231220125302.png]]
          • 1 PAM相当于所有氨基酸平均有1%发生了变化
          • 经过PAM 100次的进化，并非每个氨基酸发生了变化
          • 利用大于100PAM的时间间隔可能区分进化关系较远的同源蛋白质序列
          • PAM后面的数字，表示一种进化距离，数字越大，进化距离越远
          • PAM250是早期研究中应用最广的替换矩阵
          • JTT矩阵
      • BLOSUM替换矩阵
        • 蛋白质序列高度保守区（BLOCKS）用于构建BLOSUM矩阵
          • 优点：符合实际观测结果
          • 不足：不于进化关系挂钩
        • 假设$f_{ij}$为序列比对中氨基酸i和j对，则i和j对氨基酸所占比例$$q_{ij}=\dfrac{f_{ij}}{\sum\limits_{i=1}^{20}\sum\limits_{j=1}^{i}f_{ij}}$$
        • 在完全独立的情况下，该比例的期望值为
          $$
          e_{ij}=\left{
          \begin{aligned}
          p_i^2 \quad i=j \
          2p_ip_j \quad i\neq j \
          \end{aligned}
          \right
          .
          $$
        • 其中$p_i$和$p_j$序列分别为第i和第j个那计算所占比例
          $$
          p_i=q_i+\sum\limits_{j\neq i} q_{ij}/2
          $$
        • 矩阵单元值$s_{ij}$
          $$S_{ij} = 2\log_{2}\left( \frac{q_{ij}}{e_{ij}} \right)$$
        • 矩阵后的数字越大，表示关系越近
      • 定义了不同进化差异下氨基酸替换矩阵
      • 根据物种或序列潜在差异程度，选择对应的氨基酸替换矩阵
  • 空位罚分
    • 常数型空位罚分
      • 匹配：+8
      • 错配：-5
      • 空位延伸罚分：0
      • 空位开放罚分：-4![[Pasted image 20231221150852.png]]
    • 仿射函数空位罚分
      • 匹配：+8
      • 错配：-5
      • 空位延伸罚分：-3
      • 空位开放罚分：-4![[Pasted image 20231221151104.png]]
      • 长度为k的空位罚分$$x+k*y$$

双序列比对

  • Needleman-Wunsch算法
    • 动态规划将一个问题，拆分成一个个子问题，直到子问题可以直接解决。然后，把子问题的答案保存起来，以减少重复计算，再根据子问题答案反推，得出原问题解的一种方法![[Pasted image 20231221151920.png]]
    • 寻求两序列间距离的最小即最短距离
    • 在给定计分规则的情况下，数学上可证明能给出具有最高比对打分的对比矩阵。
    • 动态规划进行序列比对的基本思想
      • 在(i,j)处的最佳排列是在(i,j)之前最佳排列加上i和j的最佳排列
      • 重复过程直到两个序列的终点
      • ![[Pasted image 20231221153035.png]]
      • 从哪来的
      • ![[Pasted image 20231221153417.png]]
  • Smith-Waterman算法
    • 比较的时候加入0
    • 附初始值为零
    • 从最大值回溯到0
  • Semi-Global Alignment(半全局比对)
    • 只第一列初始化为0
    • 最后一列的最大值开始回溯
  • 启发式算法
  • BLAST算法
    • 应用
      • 序列比对
      • 基于序列比对，推断位置序列功能
      • 基于序列比对，寻找某物种同一家族的蛋白质序列
      • 基于序列比对，寻找物种之间的同源基因/蛋白
    • 基本流程
      • 对于待比对或检索的序列，建立词列表或哈希表![[Pasted image 20231221154527.png]]![[Pasted image 20231221154535.png]]
      • 确定搜索目标![[Pasted image 20231221154612.png]]数据库总所有包含上述词列表中词完全一样的序列
      • 对之前得到的每一条序列，在匹配区域向两端以动态规划算法向外延伸，扩展比对区域，得到高得分联配对（HSP），根据设定的玲姐比对打分值或E值，列出所有HSP![[Pasted image 20231221154647.png]]
    • 分类![[Pasted image 20231221154715.png]]
    • PSI-BLAST
      • 在蛋白质数据库中循环搜索查询蛋白质，所有前一次被PSI-BLAST发现的统计显著蛋白质序列将整合成新记分矩阵（PSSM），通过多次迭代比对，直到不再发现统计显著的新蛋白质。
        • 起始序列
        • 在全序列数据库中搜索相似的序列
        • 惊喜多序列对位排列
        • 构建一个概型（位点特异分数矩阵PSSM），并且数字描述每一个位点的保守性。
        • ![[Pasted image 20231224233639.png]]
        • 新的序列被用于对位排列
        • 多次迭代得到
          • 注释的序列信息
          • 数套多重序列对位排列
          • 由PSI-BLAST所产生的概型
          • 确定相似性的阈值
      • PSSM

    • 相似性：两条序列在同一位点上的氨基酸残基的化学性质相似
