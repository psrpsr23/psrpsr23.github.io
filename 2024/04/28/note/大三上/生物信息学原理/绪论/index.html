基于以上理论，生物学家不仅必须称为计算机学者，而且也要改变他们研究生命现象的途径。

图六：生物信息学发展
  • 纵观生物信息学的发展历史，可分为四个主要阶段
    • 萌芽期（60-70年代）
      • 以Dayhoff替换矩阵和Neelleman-Wunsch算法为代表
    • 形成期（80年代）
      • 以分子数据库和FASTA为代表
    • 基因组测序时代（90年代-2005）
      • 以模式基因组测序与BLAST为代表
    • 高通量测序时代（2005-）
      • 以第二代和代三代测序技术和基因组重测序为代表

人类基因组计划（HGP）

  • 目标：整体上破解人类遗传信息的奥秘、 ![[Pasted image 20231216113449.png]]

    图七：HGP的发展

  • 生命活动三要素

    • 物质，能量，信息
    • DNA：遗传物质，双螺旋结构
    • 基因：具有遗传效应的DNA分子片段

  • 基因组：包含细胞或生物体的全套遗传信息的全部与遗传物质

  • 人类基因组：$~3.1*10^9bp$

  • 最终目标：通过国际合作，用15年的时间（1990-2005）至少投入30亿美金，构建详细的人类基因组遗传图和物理图，确定人类DNA的全部核苷酸序列，定位约10万基因，并对其他生物进行类似的研究

    • 遗传图：基因或DNA标志在染色体上的相对位置与遗传距离
    • 物理图：指标明一些界标（例如，限制酶的切点、基因等）在DNA上的位置，图距以物理长度为单位，例如染色体的带区、核苷酸对数目等
    • 序列图：

  • 终极目标：

    • 阐明人类基因组全部DNA序列
    • 识别基因
    • 建立储存这些信息的数据框
    • 开发数据分析工具
    • 研究HGP实施所带来的伦理、法律和社会问题
      ![[Pasted image 20231216125737.png]]

  • 生物复杂性问题

    • 所有多细胞物种的编码基因数目差异不大：`2-3万
    • 物种间的细胞数量和种类差异很大
    • 生物的复杂度主要由什么来控制？
      • 基因组的转录情况
        • 编码蛋白质序列
          • 人 1-2%
          • 线虫 25%
        • 基因组转录水平
          • 人 60%
          • 线虫 70%
        • 绝大部分的转录产物是非编码RNA
      • 物种之间非编码RNA差异最大

  • ENCODE（DNA元件百科）

    • 基因组不是由孤立的基因和大量”无用DNA片段“组成的，而是一个复杂的网络系统
    • 编码基因、调控元件以及非编码RNA之间，有着复杂的相互作用，共同控制着人类的生理活动

  • ENCODE Phase II

    • 引入高通量测序技术
    • 还加入了果蝇和线虫

  • ENCODE Phase III

    • 扩展到了小鼠
    • 已经分别在人类和小鼠中生成了近100万个和超过30万个顺式调控元件的注释

  • ENCODE Phase IV

    • 研究更广泛的生物样本

  • 支持

    • CGDS（计算基因组学和数据科学计划）
      • 基因组分析工具和软件资源
      • 数据管理资源
      • 基因组信息学培训和劳动力开发
    • Alliance（基因组资源联盟）
      • 6个模式生物数据库
      • 基因本体联盟
    • GTP（基因组技术计划）
      • 核酸测序技术开发
      • 基因组技术开发
      • 合成核酸技术开发
    • GVP（基因组变异计划）
      • 在整个基因组中大规模发现和分型单核苷酸多态性（SNPs）、插入缺失（InDels）和其他形式的变异
      • 开发遗传变异和单倍型的高分辨率图谱
      • 开发用于SNP、其他形式的遗传变异、单倍型和复杂形状的大规模实验和统计分析方法
      • 开发将遗传变异与表型、疾病和功能联系起来的统计方法
    • HGRP（人类基因组参考计划）
      • 人类基因组参考中心
      • 高质量人类参考基因组
      • 基因组参考表达
      • 泛基因组信息学工具开发
      • 全基因组测序技术开发

生物信息学主要研究内容和方法

  • 主要研究内容
    • 生物学数据的收集、存储、管理与提供
    • 基因组序列信息的提取和分析
    • 分子进化和比较基因组学
    • 生物大分子结构模拟和药物设计
    • 各类高通量组学数据分析与利用
    • 生物信息分析的技术与方法研究
  • 主要研究方法
    • 基于数据挖掘的方法
    • 基于模拟分析的方法
  • 核酸蛋白质序列研究分析
    • 序列：由重复的核苷酸或氨基酸单元组成的线性高分子，具有高度有序并能完成特定生物学功能的三维结构
    • 目的：揭示序列蕴含的更高级的结构和功能信息
    • 主要思想：具有相似序列的分子，可能具有相似的三维结构和生物学功能。
    • 首要任务：提取反映结构、功能性质的序列特征
    • 主要方法：基于数据挖掘或知识发现的方法：
      • 统计方法
      • 机器学习
      • 神经网络

  • 近期热点

    • 深度学习
      • 深度学习：是一种以人工神经网络为架构，对资料进行表征学习的算法
        • 适合用来挖掘大量的、多维度数据背后的潜在规则
    • 单细胞组学
      • 单细胞测序通过单个细胞进行基因组、转录组及表观基因组水平的测序，获得相应数据并进行信息分析的技术
        • 解决用组织样本测序无法解决的细胞异质性难题，为解析单个细胞的行为、机制及其与机体的关系等提供了新方法

DNA测序技术

  • 第一代测序技术——DNA双脱氧链终止法测序（sanger）

    1. 大量复制DNA，并且加入引物
    2. 将他们分别放入带有混合氨基酸的溶液中，并且加入终止氨基酸ddNTP
    3. 进行复制反应
    4. 进行凝胶电泳
    5. 得到测序结果
    • 准确率高
    • 速度慢，成本高

  • 第二代测序技术——Illumina Solexa流程

    • 将目标DNA剪切为小片段
    • 单分子独立扩增
    • 每次只复制一个碱基(A,C,T,G)并检测信号
    • 高分辨率的成像系统
    • 速度显著提高，成本明显降低
    • 读长短

  • 第三代测序技术——单分子测序

    • 基本原理：DNA 聚合酶和模板结合，4 色荧光标记4 种碱基（即是dNTP），在碱基配对阶段，不同碱基的加入，会发出不同光，根据光的波长与峰值可判断进入的碱基类型。
    • 长读长、速度快、便携
    • 错误率较高
