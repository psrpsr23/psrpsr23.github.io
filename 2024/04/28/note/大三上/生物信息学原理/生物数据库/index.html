  • 什么是数据库
    • 用于收集、整理、储存、加工、发布和检索数据的系统
    • 是存储在计算机系统内的有结构的数据集合，是相关数据的集合。数据由数据库管理系统统一管理和维护
      • 原始数据
      • 注释
    • 一个数据库通常链接了多个数据库
  • 什么数据可以用来创建数据库
    • 核苷酸序列
      • DNA
      • RNA
    • 蛋白质序列和结构
    • 序列与结构注释
      • 基因注释
      • 功能注释
    • 代谢产物（底物、产物、酶
  • 核苷酸数据库
    • GenBank
      • 美国NCBI的数据库
      • 生信领域的一个重要的管理机构
        • 建立和管理分子生物学和生物医学领域的公共数据库
        • 发展新的分析软件
      • 默认使用GenBank flatfile格式![[Pasted image 20231220114205.png]]![[Pasted image 20231220114215.png]]![[Pasted image 20231220114657.png]]
        • ESF(序列表达标签)
          • cDNA的5或3端的序列
          • 长度200-500bp
          • 用于勘测某个生物的转录组
        • TSA
          • 非实验直接获得
          • 不同克隆的多条序列经过生物信息学软件拼接得到
          • 只能由产生原始序列的研究者递交
        • TPA
          • 第三方提供注释
        • GSS
          • 基因组短序列
          • 转座子插入位点的侧翼序列
        • STS
          • 短序列，仅在基因组中出现一次
          • 引物形式存储与数据库中
      • EMBL格式，包含的内容与GenBank相同
      • 最简单的序列格式——FASTA
        • >序列信息，注释信息
        • 具体序列
    • HTGS数据库
      • GenBank的二级数据库
      • 尚未完成测序的重叠组的序列
      • 新序列的增加速度很快![[Pasted image 20231220115437.png]]
      • 鸟枪法测序流程![[Pasted image 20231220115552 1.png]]
  • 基因组数据库
    • NCBI的另一个数据库
    • 测序完成和正在测序物种基因组序列、遗传图、物理图
    • 序列收集在GenBank
    • NCBI主页选择Genome后输入关键词
  • 其他高通量测序相关数据库
    • 单核苷酸多态性(SNP)数据库
      • 数据种类
        • SNP>99%
        • Small Insertion/deletion
      • 发现致病基因、进化分析、群体遗传分析、关联分析的遗传标记等
    • 基因组结构变异数据库
      • dbVar
      • NCBI的数据库
      • 基因组大片段的
        • 插入
        • 缺失
        • 倒位
        • 易位
    • 非编码RNA数据库
      • RNAcentral
    • 国家生物信息中心
      • 数据资源部
    • 国家基因组科学数据中心
    • 组学原始数据归档库
  • 蛋白质数据库
    • Uniport蛋白质数据库
      • 包含
        • UniPortKB
        • UniRef
        • UniParc
      • 来源于实验的有详细注释的序列和自动注释序列
      • 检索
        • 在数据库主页搜索框选择“Protein Knowledgebase”库，使用关键词检索
        • ![[Pasted image 20231220120654.png]]![[Pasted image 20231220120659 1.png]]![[Pasted image 20231220120705 1.png]]![[Pasted image 20231220120709 1.png]]
    • PIR蛋白质数据库
      • 信息整合的蛋白质序列数据库
      • 蛋白质序列分类数据库
      • 蛋白质翻译后修饰信息数据库
    • 蛋白质结构数据库
      • PDB
        • 182949个结构图
    • 蛋白质结构分类数据库
      • SCOP对已知三维结构（PDB）的蛋白质进行分类，人工分类
      • CATH利用程序自动化分类
    • 蛋白质功能与数据库
      • 一般指蛋白质序列中一段保守的区域，能够独立行使功能
      • 许多蛋白包括多个结构功能域
        • PROSITE
        • Pfam
        • SMART
        • InterPro数据库
    • 蛋白质分子互作数据库
  • 通路数据库
    • 代谢通路数据库
      • KEGG
        • 酶和代谢、信号转导通路数据库
          • 各种代谢、遗传等路径图
          • 可检索参于各种路径的基因
      • PMN
      • MetaCyc
      • Reactome
    • 代谢组学数据库
      • MetaboLights
      • HMDB
    • 其他通路数据库
      • Pathguide
  • 文献数据库
    • PubMed
      • 美国国家医学图书馆的数据库
      • 医学、分子生物学、基础生物学
      • 5400多种刊物，来源于80多个国家
      • 文献年限1965年至今
      • 提供摘要，全文链接

向数据库提交和修改数据

  • 发表文章需要提交序列
  • 数据库中的数据由大家无偿提供、共同享用
  • GenBank
    • 主页->菜单->BankIt
    • 使用Sequin方法提交序列
    • 修改需要邮件联系
