基因芯片

  • 转录组
    • 一个活细胞所能转录出来的所有RNA的总和
    • 是解析生物学机制和基因功能的重要途径
  • 概念
    • 固定着大规模的DNA片段，用于检测整个转录组内成千上万个基因的mRNA表达水平的变化的微型器材
  • 流程
    • 提取RNA
    • 反转录
    • 扩增
    • 标记
    • 杂交
    • 扫描
    • 获得原始数据
  • 优势
    • 稳定可靠
    • 积累了大量不可重新获取的数据
  • 局限
    • 智能检测已知或确定性的序列
    • 无法检测新发现、未放置到芯片上的基因
    • 有部分探针的信号可能会收到非特异性杂交及个体序列差异的影响

表达谱数据库介绍

  • GEO数据库
    • 包含提交到NCBI的各种物种的基因表达谱数据

芯片数据前处理

  • 数据过滤
  • 背景纠正
    • 从总体信号中减去噪声信号，从而得到真实信号
    • PM和MM组成一组探针，MM是检测非特异杂交信号，应该比PM小，但实际上33%的MM>PM
  • 均一化
    • 几乎所有高通量技术都涉及该数据处理过程
    • M-A plot![[Pasted image 20240115160429.png]]
    • median normalization(中位数均一化)
    • Loess normalization(成对样本均一化)
    • quantile normalization(分位数均一化)
      • 根据列排序
      • 计算每行平均值
      • 按照原来列的排序重新排序![[Pasted image 20240115154554.png]]
  • 探针集信号汇总![[Pasted image 20240115154925.png]]
    • Average Difference method(MAS 4.0)
      • 30%-40%探针过滤，可能产生负值
    • MAS 5.0
    • Li&Wong(dChip)
    • RMA
      • Use Median Polishing
      • 每行减去行中位数
      • 每列减去列中位数
      • 直到行列中位数都为0
      • 用原始矩阵减现在的矩阵
  • 补缺失值
  • 批次效应

基因表达谱差异表达分析

  • 基于一些统计学模型，对不同样本处理下的基因表达差异进行分析
  • 要先对样本进行相关性分析或者主成分分析
  • 差异表达倍数（Fold change）$$\frac{AVG(X)}{AVG(Y)}$$![[Pasted image 20240115161019.png]]
  • 衡量置信度
    • 正态性检验Test Normality
      • t-test
      • 非参数检验non-parametric test
    • 威尔克逊秩和检验Wilcoxon Rank Sum
    • 差异表达线性模型Linear Model![[Pasted image 20240115161553.png]]
    • 一般的t检验Ordinary t-tests
    • 两样本比较的方差估计Variance Estimates
  • p-value![[Pasted image 20240115161816.png]]
  • 多重假设检验Multiple Hypothesis Testing
    • p值只对一次实验有效
    • 组错误率FWER，在一系列实验中至少犯一次错的概率![[Pasted image 20240115162232.png]]
      • 往往会过于严苛
    • TPR(sensitivity)=TP/(TP+TN)
    • TNP(specificity)=TN/(TN+FP)
    • FPR=FP/(TN+FP)[P-value]
    • FDR=FP/(TP+FP)[往往大家更关心]
      • 鉴定n个显著差异表达基因中包含错误个数<5%![[Pasted image 20240115162722.png]]
      • Q-value=p(k)*m/k

基因本体论及富集分析

  • 基因本体论GO
    • GO注释：用标准化的术语来全面描述基因及其产物的功能和属性
      • 细胞组分CC
      • 分子功能MF
      • 生物过程BP
  • 富集分析
    • 关注一组基因
    • 挖掘在数据库中我们要研究的生物学问题具有显著相关性的基因功能类别
    • 过表达分析（ORA）![[Pasted image 20240115170449.png]]检验差异表达基因是否与某个类别或者途径相关
    • 功能集打分
      • 根据各基因在不同处理中表达差异的大小将基因排序
      • 计算基因集富集分数（ES）![[Pasted image 20240116001538.png]]
      • 通过随机模拟背景分布估计富集的显著度
      • 使用多重假设
