质量性状的概念及遗传特点

  • 在一个群体内表现不连续变异的形状，相对性状间有质的差异
    • 表现型和基因型的变异不连续
    • 相对性状之间存在显隐性
    • 不易受环境条件的影响而发生变异
    • 眼睛观察，文字记载描述，符合孟德尔定律

数量性状的概念及遗传特点

  • 在一个群体内表现为连续变异的性状，相对性状间没有质的差异
    • 不存在显性与隐性的关系
    • 对环境条件反应敏感
    • 依靠群体，使用统计方法
  • 受主效基因转换
  • 阈性性状

纯系学说

  • 内容
    • 一个基因型纯合个体自交产生的后代，其后代群体的基因型也是纯合的

分子标记

是以个体间遗传物质的核苷酸序列变异为基础的遗传标记

  • 第一代分子标记
    • DNA限制片段长度多态性RFLP
      • 以Southern杂交为核心，由限制性内切酶识别序列发生变异，导致酶切位点数量发生变化
  • 第二代分子标记
    • DNA随机扩增多态性RAPD
      • 利用随机引物对目的基因组DNA进行PCR扩增，产物经电泳分离后显色，分析扩增产物DNA片段的多态性
    • 简单重复序列SSR
      • 以1-6个碱基为重复单元的串联重复序列的长度变异而产生的多态性
    • 扩增片段长度多态性AFLP
      • 基于PCR反应的一种选择性扩增限制性片段的方法
  • 第三代分子标记
    • 单核苷酸多态性SNPs
      • 在基因组水平上由单个核苷酸的变异所引起的DAN序列多态性
    • SNP标记检测方法
      • 是一种DNA芯片，通过杂交的方法检测个体间已知多态性位点的基因型
      • 竞争性等位基因特异性PCR KASP
        • 基于引物末端碱基的特异匹配来对SNP进行分型，用于检测大量样本的少量标记
        • 基因组重测序
        • 转录组测序
        • GBS
          • 检测酶切位点附近的序列、

QTL定位原理

  • QTL：代表染色体上影响数量性状的某个区段，区段内可能会有一个甚至多个影响数量性状的功能基因
  • QTL定位：检查分子标记与QTL间的连锁关系，并估算QTL的表型效应。
    • 实现对功能基因的定位
  • 构建合适的作图群体
    • 亲本的选择
    • 群体大小
    • 分离群体的类型
